# nve-rsync-monitor-project

Monitoring project for rsync data synchronization between Source and Target server.
 
## Modules
- [nve-rsync-host-source](https://github.com/drvicx/nve-rsync-host-source) : Collects directories Data (files count, size total) on Source Server
- [nve-rsync-host-target](https://github.com/drvicx/nve-rsync-host-target) : Collects directories Data (files count, size total) on Target Server
- [nve-rsync-host-monitor](https://github.com/drvicx/nve-rsync-host-monitor) : Monitors & Compares sync result between Source and Target server

## Linked Project
- [nve-rsync-host-monitor-test](https://github.com/drvicx/nve-rsync-host-monitor-test) : Simple Python REST API Service App for tests
