# nve-rsync-monitor-project

Monitoring project for rsync data synchronization between Source and Target server.
 
## Modules
- nve-rsync-host-source  :: Collects directories info on Source Server
- nve-rsync-host-target  :: Collects directories info on Target Server
- nve-rsync-host-monitor :: Monitors & Compares sync process between Source and Target server
